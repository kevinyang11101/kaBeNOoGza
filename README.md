# P201房屋信息可视化系统 python679

## 项目概述

P201房屋信息可视化系统是一个基于Python技术的项目，结合了Django框架和MySQL数据库，致力于提供房屋信息的注册、预测和管理等功能。

## 技术栈

- 后端：Python
- 框架：Django
- 数据库：MySQL

## 功能模块

### 用户注册

- 系统仅对注册用户开放。

### 首页模块

- 展示最新的房价信息和房屋详情。

### 房价分析

- 自动爬取安居客、58同城等平台的房价数据。
- 精准预测房价趋势。

### 房屋管理

- 保存和管理大量房屋信息。
- 支持信息的增删改操作。

### 房屋饼状图

- 直观展示不同房型的比例。
